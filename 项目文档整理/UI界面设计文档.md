# 权利卫士自动化 - UI界面设计文档

[← 返回README](../README.md)

---

## 🎨 设计理念

本项目采用**Material Design**设计规范,打造简洁、美观、易用的Android应用界面。

### 核心设计原则
- ✨ **简洁明了** - 界面布局清晰,功能一目了然
- 🎯 **统一配色** - 淡蓝色主题,白色背景,视觉舒适
- 📱 **响应迅速** - 操作反馈及时,用户体验流畅
- 🔧 **易于维护** - 模块化设计,便于后续扩展

---

## 🎨 配色方案

### 主色调
- **淡蓝色**: `#64B5F6` - 按钮、强调元素
- **深蓝色**: `#42A5F5` - 按钮按下状态
- **浅蓝色**: `#90CAF9` - 辅助元素

### 背景色
- **纯白色**: `#FFFFFF` - 主背景
- **浅灰色**: `#F5F5F5` - 卡片背景

### 文字颜色
- **深灰色**: `#333333` - 主要文字
- **中灰色**: `#666666` - 次要文字
- **浅灰色**: `#999999` - 提示文字
- **纯白色**: `#FFFFFF` - 按钮文字

### 状态颜色
- **成功**: `#4CAF50` (绿色)
- **警告**: `#FF9800` (橙色)
- **错误**: `#F44336` (红色)
- **信息**: `#2196F3` (蓝色)

## 📱 界面结构

### 1️⃣ 主界面 (MainActivity)

#### 功能布局
```
┌─────────────────────────────┐
│   权利卫士自动化      [设置] │  ← 标题栏(淡蓝色)
├─────────────────────────────┤
│                             │
│  ┌───────────────────────┐  │
│  │  当前状态:            │  │
│  │  ● 空闲               │  │  ← 状态卡片
│  │  等待开始自动化任务    │  │
│  └───────────────────────┘  │
│                             │
│  ┌───────────────────────┐  │
│  │    开始自动化          │  │  ← 主按钮(淡蓝色)
│  └───────────────────────┘  │
│                             │
│  ┌───────────────────────┐  │
│  │    停止自动化          │  │  ← 次要按钮(白底蓝边)
│  └───────────────────────┘  │
│                             │
│  ┌───────────────────────┐  │
│  │    查看日志            │  │  ← 次要按钮
│  └───────────────────────┘  │
│                             │
│  ┌───────────────────────┐  │
│  │  使用说明              │  │
│  │  1. 确保已开启无障碍... │  │  ← 说明卡片
│  │  2. 确保已安装权利卫士  │  │
│  └───────────────────────┘  │
│                             │
└─────────────────────────────┘
```

#### 核心元素
- **标题栏**: 淡蓝色背景,白色文字,右侧设置图标
- **状态卡片**: 白色卡片,圆角12dp,阴影4dp
- **状态指示器**: 圆形指示灯,根据状态变色
- **主按钮**: 淡蓝色背景,白色文字,圆角8dp,高度56dp
- **次要按钮**: 白色背景,淡蓝色边框,圆角8dp,高度48dp
- **说明卡片**: 浅灰色背景,提示性信息

### 3️⃣ 日志界面 (LogActivity)

#### 功能布局
```
┌─────────────────────────────┐
│  [←]  运行日志      [导出]  │  ← 标题栏(淡蓝色)
├─────────────────────────────┤
│                             │
│  ┌───────────────────────┐  │
│  │ [2024-01-19 10:30:15] │  │
│  │ 系统启动               │  │
│  │ [2024-01-19 10:30:16] │  │
│  │ 检查权限...            │  │  ← 日志显示区域
│  │ [2024-01-19 10:30:17] │  │  (等宽字体)
│  │ 无障碍服务已启用        │  │
│  │ ...                   │  │
│  └───────────────────────┘  │
│                             │
│  [清除日志]    [导出日志]    │  ← 底部按钮
│                             │
└─────────────────────────────┘
```

#### 核心元素
- **日志区域**: 浅灰色背景,等宽字体,可滚动
- **时间戳**: 统一格式显示
- **清除按钮**: 次要按钮样式
- **导出按钮**: 主按钮样式

### 4️⃣ 悬浮窗 (FloatingWindow)

#### 功能布局
```
┌─────────────────────┐
│      ────           │  ← 拖动手柄
│                     │
│  ● 空闲             │  ← 状态显示
│                     │
│  [开始] [停止] [×]  │  ← 控制按钮
│                     │
└─────────────────────┘
```

#### 核心元素
- **圆角卡片**: 24dp圆角,8dp阴影
- **拖动手柄**: 灰色横条,便于拖动
- **迷你按钮**: 紧凑布局,快速操作
- **关闭按钮**: 右侧×图标

## 🎯 交互设计

### 按钮状态
1. **正常状态**: 淡蓝色背景
2. **按下状态**: 深蓝色背景
3. **禁用状态**: 灰色背景,半透明

### 状态指示
- **空闲**: 灰色圆点
- **运行中**: 蓝色圆点
- **录屏中**: 绿色圆点
- **错误**: 红色圆点

### 动画效果
- **按钮点击**: 涟漪效果
- **页面切换**: 滑动过渡
- **状态变化**: 淡入淡出

## 📐 尺寸规范

### 间距
- **页面边距**: 16-20dp
- **卡片间距**: 16-24dp
- **元素间距**: 8-12dp

### 圆角
- **卡片圆角**: 12dp
- **按钮圆角**: 8dp
- **悬浮窗圆角**: 24dp

### 阴影
- **卡片阴影**: 4dp
- **悬浮窗阴影**: 8dp

### 字体大小
- **标题**: 20sp (粗体)
- **正文**: 16sp
- **次要文字**: 14sp
- **提示文字**: 12sp
