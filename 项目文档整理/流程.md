# 权利卫士取证自动化 V5.3 当前完整流程

## ① 用户输入
- 打开APP
- 粘贴取证信息（原创名称-抖音:侵权人名称 + 侵权链接 + 商品信息等）
- 点击开始

## ② 打开侵权视频
- 自动跳转抖音侵权视频
- 视频自动播放3秒

## ③ 启动录屏取证
- 自动打开权利卫士 → 点击【录屏取证】
- 自动填写备注 → 点击【开始录屏取证】
- 处理录屏权限弹窗 → 点击【立即开始】
- 点击【开始取证】→ 跳转抖音

## ④ 抖音作者设置页取证
- 点击底部【我】→ 右上角【更多】→ 【设置】
- 下滑 → 点击【资质证照】→ 📸 截图
- 返回 → 点击【营业执照】→ 📸 截图
- 返回"我"页面

## ⑤ 订单与资质规则取证
- 点击【我的订单】→ 右上角【更多】→ 📸 截图
- 点击【资质规则】→ 📸 截图 → 下拉到底部
- 返回"我"页面

## ⑥ 视频播放、评论截图与链接分享
- 点击【观看历史】→ 点击侵权视频播放
- 自动截图（4-5张）
- 暂停 → 打开评论区 → 滚动 → 截图（1-3张）→ 关闭评论区
- 📤 分享视频链接给QQ【我的电脑】→ 发送
- 检测是否有小黄车：
  - **无小黄车** → 跳至 ⑧ 作者主页流程
  - **有小黄车** → 进入 ⑦ 小黄车取证

## ⑦ 小黄车取证（V5.3）

**进店取证（第一步，V5.3新增）**
- OCR查找【进店】→ 📸 截图（商品详情页）→ 点击进入店铺
- OCR等待加载（"全部商品"）→ 📸 截图（店铺主页）
- 点击店铺信息卡片(400,386) → 📸 截图（店铺详情）
- OCR找【资质证照】→ 点击 → 📸 截图（资质证照）
- 返回×3 → 回到商品详情页

**选品带货（第二步）**
- 点击三个点(996,180) → 等待弹窗 → 📸 截图（更多弹窗）
- 点击【选品带货】(576,2072) → 等待加载 → 📸 截图（选品带货页）

**联系商家（第三步）**
- 点击【联系商家】(256,2176) → 等待聊天页加载
- 点击右上角电话按钮(898,188) → 📸 截图（联系商家电话）
- 返回×1 → 回到选品带货页

**带货数据90天（第四步）**
- 检测千川数据（无障碍节点优先 + OCR兜底）
- 有千川数据：📸 截图 → 下滑找【带货数据】→ 点击【30天】→ 选【近90天】→ 📸 截图

**创作灵感（第五步）**
- 在创作灵感中按封面Key查找侵权视频
- 点击视频 → 播放 → 📸 截图
- � 分享该视频链接给QQ【我的电脑】→ 发送

**带货达人（第六步）**
- 在带货达人Tab按侵权人名称匹配账号
- 点击账号 → 点击【近7日动态】→ 选【近30日】→ 📸 截图
- 在视频Tab按封面Key查找侵权视频 → 点击播放
- 展开数据面板(666,1431) → 隐藏文案抽屉(540,1561→2346) → 📸 截图
- 等待视频播放完毕 → 返回视频列表 → 点击达人头像 → 进入达人主页

## ⑧ 作者主页取证
- 检测蓝V/认证标签（店铺授权号 / 店铺账号 / 官方账号 / 企业账号等）
- 找到 → 点击 → OCR等待加载 → 📸 截图（认证详情）
- 未找到 → 跳过
- ✅ 取证完成 → 返回权利卫士

## ⑨ 停止录屏与生成PDF
- 点击【停止录屏取证】
- 自动生成取证PDF
  - 命名：原创名称-抖音：侵权人账号名称.pdf
  - 保存到手机下载目录
- ✅ 取证完成！
